# PROJECT-DOCTOR-AI

Project Doctor AI is a tool that helps students improve their programming projects.

## Problem
Many students struggle to debug code and improve their project ideas.

## Solution
Project Doctor AI analyzes code and suggests improvements, best practices, and possible features.

## Features
- Code analysis
- Improvement suggestions
- Beginner-friendly feedback
- Open-source friendly

## Tech Stack
Frontend: HTML, CSS  
Backend: Python  
Tools: Git, GitHub

## How to Run
1. Clone the repository
2. Run backend/app.py
3. Open frontend/index.html

## Future Improvements
- AI-based code analysis
- GitHub issue recommendations
- Project scoring system
